# Разбраковка линейки фотодетекторов

## Тестирование по темновому сигналу
**Цель тестирования** - определение фотоячеек, входящих в состав линейки фотодетекторов, у которых отклонение темнового сигнала от среднего по линейке превышает заданный порог.

**Входные параметры тестирования:**
- времени экспозиции $\tau_{0} = 10$, мс;
- отсутствие светового потока: $i_{l} = 0$;
- максимальный разброс $\delta_{\alpha}$ (например $10\%$);
- маска дефектных фотоячеек $M$ - массив размером $1 \times n$, заполненный $0$.

**Выходные параметры тестирования:**
- значение темнового сигнала $\bar{U}$;
- разброс темнового сигнала $\delta$;
- маска дефектных фотоячеек $M$.

**Алгоритм тестирования:**
1. Измеряется темновой сигнал $U$ в отсутствие светового потока $i_{l} = 0$ за время экспозиции $\tau_{0} = 10$, $мс$: $$U = U_{b} + \tau_{0} \cdot i_{d}$$
2. Рассчитывается среднее значение темнового сигнала: $$\bar{U} = \frac{1}{n}\sum_{i: M_{i} = 0}{U_{i}}$$
3. Рассчитывается разброс темнового сигнала: $$\delta=\frac{U_{\max} - U_{\min}}{\bar{U}},$$ где $U_{\min}$ и $U_{\max}$ рассчитываются с учетом маски дефектных фотоячеек $M$.
4. Если рассчитанный разброс превышает установленный порог $\delta > \delta_{\alpha}$, то итеративно вычисляется дефектные фотоячейки. Фотоячейка с наибольшим отклонением от рассчитанного среднего значения $\bar{U}$ маскируется: $M_{i} = 1$. Повторяются расчеты 2-3 пока $\delta > \delta_{\alpha}$.
